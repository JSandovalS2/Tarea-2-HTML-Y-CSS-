¿Qué es HTML y CSS?

HTML es el lenguaje de programación con el que se “escribe” la inmensa mayoría de páginas web, es decir, es el lenguaje que se emplea para la creación de los websites a los que se accede a través de una conexión a internet.


HTML fue evolucionando, también se comenzó a incluir información relativa al aspecto de esos contenidos tales como colores, márgenes, tipos de letra, etcétera. Es así como apareció CSS, un mecanismo que permite mantener separados los contenidos que se definen mediante HTML, del aspecto (diseño) que deben mostrar tales contenidos.
